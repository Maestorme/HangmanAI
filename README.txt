Author: Karthick Shankar
Email: shankak@purdue.edu

Programming Language: Python3

Required files:
hangman.py
wiki-100k.txt


Modules used:
string (built-in)
requests
sys (built-in)

To install the requests module, type

    python -m pip install requests

To run this from the command line, type

    python hangman.py

Commandline options:
    -v : VERBOSE
         Will print some messages while running like letter frequency, current word etc.
